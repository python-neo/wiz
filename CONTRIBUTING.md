# Contributing

Thanks for contributing to Wiz. This is an early-stage project, so feedback and
small improvements are especially valuable.

## Development Setup

- Install Python 3
- Create a virtual environment
- Work from the repo root

Suggested commands:

```text
python -m venv .venv
.\.venv\Scripts\activate
```

If you are on macOS or Linux:

```text
python3 -m venv .venv
source .venv/bin/activate
```

## Running Tests

Test tooling is not finalized yet. If tests exist, run them with:

```text
python -m pytest
```

## Style

- Keep changes small and readable
- Match the existing style in the file you are editing
- Prefer clear, explicit names over cleverness

## Submitting Changes

- Make focused commits
- Update `CHANGELOG.md` for user-facing changes
- Describe the intent in your PR or message

## Questions

Open an issue or start a discussion if you are unsure about design choices. Early
alignment is welcome.
