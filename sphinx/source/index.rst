Wiz Documentation
=================

Wiz is a hacker-vibe command-line shortcut manager. It stores frequently used
commands as aliases and lets you run, replace, delete, and export them quickly.
The CLI uses a lightweight JSON config file and provides colored output and
help text for a clean terminal experience.
