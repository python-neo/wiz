# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

- 

## [0.6.3] - 2026-04-03

### Added

- Linux-style alias lines for `export`
- Shell-aware execution for `run` on Windows and Linux
- Flags can appear before or after the command
- Sphinx docs release updated to 0.6.3

## [0.6.0] - 2026-03-28

### Added

- `export` command for shell-friendly alias output
- `--no-color` flag for disabling colors
- `--version` flag for printing version

## [0.5.0] - 2026-03-28

### Added

- `run` command with confirmation when a command is missing

## [0.4.0] - 2026-03-28

### Added

- `delete` and `replace` commands
- Rich console output and themed help text

## [0.3.0] - 2026-03-28

### Added

- `add` command with config persistence
- `list` reads from `config.json`

## [0.2.0] - 2026-03-27

### Added

- Added `list` command and config file scaffolding

## [0.1.1] - 2026-03-27

### Added

- MIT LICENSE file

## [0.1.0] - 2026-03-27

### Added

- Initial CLI entry point with help output
- Command parsing scaffolding

## [0.0.1] - 2026-03-25

### Added

- Project structure
- Initial documentation files
